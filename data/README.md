# Dataset

This project uses the **CICIDS 2017** dataset (Friday DDoS traffic).

- Source: https://www.unb.ca/cic/datasets/ids-2017.html
- File used: `Friday-WorkingHours-Afternoon-DDos.pcap_ISCX.csv`
- Raw data is stored in: `s3://aws-ids-platform/raw/`
- Processed data is stored in: `s3://aws-ids-platform/processed/`

Do not commit raw data files to this repository.
